# Monitoramento de Greves dos Correios

Um projeto desenvolvido por mim para acompanhar as paralisações dos Correios em tempo real.

## Sobre o Projeto

Este site é dedicado a auxiliar importadores e compradores de produtos da China, oferecendo informações atualizadas sobre greves e paralisações nos serviços dos Correios que podem afetar as entregas.

## Tecnologias

- React
- Vite
- CSS
- Typescript

## Como Iniciar

1. Clone o repositório:
```bash
git clone https://github.com/2jjj/greve.git
```

2. Entre no diretório:
```bash
cd greve
```

3. Instale as dependências:
```bash
npm install
```

4. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

O projeto estará disponível em `http://localhost:8080`
